# minivim

## info

A "mini" implementation of vim :3

It uses `VT100` escape characters (I will implement `ncurses` in the future probably).

## status

- [x] enter terminal raw mode
- [x] basic input / output
- [x] view files
- [x] edit files
- [x] syntax highlighing
- [x] search feature
- [ ] basic vim features
